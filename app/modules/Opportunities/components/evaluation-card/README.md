# Componente `<evaluation-card>`
Cartão de avaliação e suporte
  
## Propriedades
- *Entity **entity*** - Entidade

### Importando componente
```PHP
<?php 
$this->import('evaluation-card');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<evaluation-card :entity="entity"></evaluation-card>

```
