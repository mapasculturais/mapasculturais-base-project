# Componente `<registration-actions>`
Componente de ações em uma inscrição
  
## Propriedades
- *Entity **entity*** - Entidade

### Importando componente
```PHP
<?php 
$this->import('registration-actions');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<registration-actions :entity="entity"></registration-actions>

```
