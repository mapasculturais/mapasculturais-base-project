# Componente `<opportunity-phase-header>`
Exibe um header para a fase

## Propriedades
- *Entity **phase*** - Fase

### Importando componente
```PHP
<?php 
$this->import('opportunity-phase-header');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<opportunity-phase-header :phase="phase"></opportunity-phase-header>
```
