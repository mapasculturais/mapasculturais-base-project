# Componente `<entity-profile>`
Componente de edição da imagem de perfil da entidade
  
## Propriedades
- *Entity **entity*** - Entidade

### Importando componente
```PHP
<?php 
$this->import('entity-profile');
?>
```

### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<entity-profile :entity="entity"></entity-profile>
```
